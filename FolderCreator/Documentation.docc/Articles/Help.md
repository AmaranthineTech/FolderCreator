# Help

## You can get help by running the following commands
```shell
 folderCreator -h
 folderCreator -help
```
 OR
```shell
 man folderCreator
```

> Important: It is assumed that the man page for the script has been deployed before use. For more information on how to deploy the man page visit [Shell scripting in macOS â€“ Part 10: Distribution](https://arunpatwardhan.com/2023/02/02/shell-scripting-in-macos-part-10-distribution/). You can also view the <doc:Installation> article.

## You can also view the log file for the same at

```shell
~/Library/Logs/folderCreator_log_v1-8.log
```

> Tip: You can view the log file by opening the Console app. Since the log is placed in a standard location it should automatically appear in the app.
